FLL Competition Suite

This application is for use in the FIRST LEGO LEAGUE competition. It is written in pure Python and is web-rendered using Flask. The application is split into 2 parts


The first part is the Schedule Generator, It is built to take a list of teams and assign them a room to stay in and then from there create both a Robot Game schedule and a Judging schedule these are then checked next to each other to ensure that there is no overlap. It will then check for any rooms that are shared between teams and then ensure that there is no improper overlapping


The second part is the ScoreBoard, It is built using flask, jinja and just a little bit of ECMAScript. This lightweight application stores in team data from an excel sheet. It then sorts that data to create a score list of every team.


Currently I am looking to re-implement this in Node, Electron, and Angular, development of this version (4.0) will start in May 2020 for use in the following fall season.
