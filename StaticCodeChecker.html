Static_Code_Checker

Static_Code_Checker is a tool to be used in classroom environments for both students and teachers. The application renders a front end by using Python and flask to host it the same way that an normal website would be setup but just running locally

Static_Code_Checker uses python to compile the code by getting into the command line, this allows it to compile a students code and reference it back to a selected output file provided by the teacher. This allows students to better test their code.
